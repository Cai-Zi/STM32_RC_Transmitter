## 1.材料清单 
- STM32F103C8T6蓝色开发板1（黑色板也可以） 
- USB转TTL模块1 
- ST-LINK V2仿真器下载器1（调试STM32性价比极高） 
- OLED屏幕（4管脚）1 
- 航模遥控器所用的遥杆2 
- 两段/三段钮子开关>=2 
- 3.7V锂电池1 
- 升压充电模块*1

## 2.安装Keil5(MDK) 
- 参考链接进行安装：https://blog.csdn.net/weixin_42911200/article/details/81590158 
- 注意要安装Keil.STM32F1xx_DFP.2.3.0.pack支持包，因为我们要用STM32F103C8T6芯片的库函数编写。
